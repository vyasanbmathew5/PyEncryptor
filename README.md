# PyEncryptor
**Simple Python Code Encryptor.**


### Supported Language:
- **python 2.x**
- **python 3.x**

### Installation :

```bash
apt install git python -y
git clone https://github.com/vyasanbmathew5/PyEncryptor
cd PyEncryptor
python encode.py
```
